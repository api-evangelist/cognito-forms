# Cognito Forms

Cognito Forms is an online form builder with a REST API for managing forms, retrieving entries, updating submissions, and integrating form data into business applications. The API uses API key bearer token authentication and supports operations for forms, entries, and documents, with OData support for analytics tool integration via Power BI, Tableau, and Excel.

## API

The Cognito Forms REST API enables integration with existing systems without third-party tools. It supports:

- **Forms**: Get forms, get form fields, get form schema, set form availability
- **Entries**: Get entry, create entry, update entry, delete entry
- **Documents**: Get documents associated with entries
- **OData Service**: Sync entry data with analytics tools (Power BI, Tableau, Excel, any OData v4 consumer)

Authentication uses API keys as bearer tokens in the Authorization header. API keys are generated by organization owners/administrators in organization settings. Up to five API keys per integration can be created.

API access is available on Pro, Team, and Enterprise plans.
